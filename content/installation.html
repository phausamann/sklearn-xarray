Installation¶

The package can be installed with pip:

$ pip install sklearn-xarray

or with conda:

$ conda install -c phausamann sklearn-xarray

Testing¶

To run the unit tests, install pytest and run:

$ pytest

Building the docs¶

To build the documentation, install sphinx, sphinx-gallery, sphinx_rtd_theme and numpydoc and run:

$ make -C docs html
